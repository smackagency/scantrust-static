Infographic: QR Code Usage Around the World

This infographic is the first in a series about the simple, but mighty useful, QR code.

Scantrust active brand protection, supply chain awareness, and consumer engagement work with QR codes. Especially secure QR codes printed on product packaging. Below, you can find an infographic contacting data on QR code usage around the world.

 

Five years ago, QR code usage was still relatively small though steadily growing in all regions. Since then, global QR code usage hasn’t slowed down. On the contrary, it has increased and diversified in surprising ways. Essential activities, like paying for groceries and lunch, have made the QR code something that people naturally interact with without giving the humble QR code a second thought. That’s a good thing. We’ve prepared some data from the last five years on QR code usage.
